# Сайт-визитка

Сайт-визитка Евгения и Иры — АИ-решения для проектировщиков.

## Состав
- Герой-блок: кто мы, что предлагаем
- Услуги и АИ-решения
- О нас
- Форма обратной связи (через Formspree)

## Запуск
Открыть `index.html` в браузере. Для разработки удобно расширение
**Live Server** в VS Code — даёт автообновление при сохранении.

## Структура
- `index.html` — главная
- `css/` — стили
- `js/` — скрипты
- `images/` — фотографии и лого
- `памятки/` — долгосрочные факты проекта (домен, ID Formspree и т.д.)
- `CLAUDE.md` — инструкции для Claude Code
