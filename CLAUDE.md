# Сайт-визитка

## О проекте
Сайт-визитка пары — Евгений и Ира. Презентация АИ-решений для проектировщиков:
готовые ИИ-ассистенты под разделы проектной документации (АР, КЖ, ОВ, ВК, ЭОМ),
с базой знаний из нормативных документов РФ.

## Состав
- Герой-блок: кто мы, что предлагаем
- Услуги и АИ-решения: бот по КЖ, расчёт стальной балки, ассистенты под разделы ПД
- О нас: опыт в проектировании, методология
- Форма обратной связи

## Технологии
HTML5 + CSS3 + минимум JS, без фреймворков. Форма обратной связи — через
сервис Formspree (без своего бэка). Хостинг — на потом (GitHub Pages /
Netlify / Cloudflare Pages — все бесплатные).

## Правила работы

- Общаться на русском
- Объяснять простым языком, без жаргона
- Перед значительными изменениями (рефакторинг, удаление, миграции) спрашивать
- После значимых изменений предлагать коммит
- Комментарии в коде на русском
- Не усложнять — выбирать простые решения, без лишних фреймворков
- Не выдумывать факты — прямо говорить «не знаю»
- Папка `памятки/` — долгосрочные факты проекта. Читать при старте сессии
- Ритуал безопасности: задача → проверка → коммит → проверка секретов → push
- Для рискованных задач — коммит-страховка ДО задачи

## Память проекта
В корне есть папка `памятки/` — там лежат долгосрочные факты: домен, хостинг,
ID формы Formspree, выбранные сервисы, цветовая палитра, шрифты. **При старте
новой сессии обязательно прочитай содержимое этой папки.**

## Структура файлов
- `index.html` — главная страница (все 4 блока на одной странице)
- `css/style.css` — стили
- `js/main.js` — валидация формы, плавный скролл по якорям
- `images/` — фотографии, лого
- `.env.example` — образец переменных (для справки, не секреты — браузер `.env` не читает)

## Запуск
Открыть `index.html` в браузере двойным кликом — для проверки локально достаточно.
Для нормальной разработки в VS Code — расширение Live Server (правая кнопка →
"Open with Live Server"), даёт автообновление при сохранении.
